# Execution Engine

Implement secure Docker-based execution.

Workflow:

1. Receive code from frontend
2. Create temporary execution directory
3. Store source code file
4. Launch Docker container
5. Compile if required
6. Execute code
7. Capture stdout
8. Capture stderr
9. Return results
10. Destroy container
11. Remove temporary files

Never execute code directly on host machine.
